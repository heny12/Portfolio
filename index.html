PROJECTS

Tectonomap





Tectonomap is an interactive map of the world’s recorded significant earthquakes and volcanic eruptions. It is meant to be used as a tool for learning and exploring Earth’s recent geologic history. Tectonomap uses the NOAA Significant Volcanic Eruption Database and Significant Earthquake Database to pull its data. It uses the Google Maps Javascript api and visualization library for the map layers.
